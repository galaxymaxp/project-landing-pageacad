# Speedlight Pro Landing Page

Responsive single-page product landing page for **Speedlight Pro**, a fictional premium wireless gaming mouse built for competitive FPS players.

## Project Overview

This project was created using only HTML, CSS, and vanilla JavaScript. The design follows a dark esports-inspired direction with a polished hero section, product storytelling, spec highlights, interactive controls, and a mobile-friendly navigation system.

## Features

- Semantic HTML5 structure with `header`, `nav`, `main`, `section`, and `footer`
- Sticky navigation with smooth scrolling links
- Responsive layout built with CSS Grid and Flexbox
- Premium dark gaming visual style with a CSS-built product mockup
- Mobile navigation toggle with accessible `aria-expanded` behavior
- Interactive DPI selector with live content updates
- Active navigation state changes based on the visible section
- CTA feedback toast for extra page interactivity

1. Clone or download the repository.
2. Open `index.html` in a browser.

No frameworks, build tools, or dependencies are required.
